## ğŸ› ï¸ Langkah Cepat (Vercel)

Vercel adalah cara paling mudah dan gratis untuk pemula.

1.  **Siapkan GitHub**: Pastikan kode Frontend kamu sudah di-upload (push) ke GitHub.
2.  **Koneksikan Vercel**: 
    - Buka [Vercel.com](https://vercel.com).
    - Login menggunakan akun GitHub.
3.  **Import Proyek**: 
    - Pilih repository yang berisi kode Frontend kamu.
    - Klik tombol **Deploy**.
4.  **Selesai!**: Kamu akan mendapatkan link khusus untuk websitemu.
