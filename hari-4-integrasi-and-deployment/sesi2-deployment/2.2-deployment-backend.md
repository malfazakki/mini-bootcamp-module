1.  **Siapkan Kode**: Pastikan file `server.js` kamu sudah di-upload ke GitHub.
2.  **Sesuaikan Port**: Pastikan di kode `server.js` kamu ada baris ini:
    ```javascript
    const port = process.env.PORT || 3000;
    ```
3.  **Koneksikan Render**:
    - Buka [Render.com](https://render.com).
    - Pilih **"New Web Service"** dan hubungkan ke GitHub.
4.  **Setting Awal**:
    - **Build Command**: `npm install`
    - **Start Command**: `node server.js`
5.  **Klik Deploy**: Tunggu 2-3 menit sampai muncul tulisan **"Live"**.
