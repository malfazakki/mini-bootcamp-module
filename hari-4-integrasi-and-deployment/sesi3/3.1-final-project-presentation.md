**Fitur:**
- Tambah pengeluaran (nama, jumlah, kategori, tanggal)

- Filter by kategori

```sql

    category VARCHAR(50),
    date DATE,
    created_at TIMESTAMP DEFAULT NOW()
);
```

### **2. Contact Manager (CRM Mini)**
**Fitur:**
- Tambah kontak (nama, email, phone, company)
- Edit & hapus kontak
- Search kontak
- Kategorisasi (client, vendor, personal)

**Database:**
```sql
CREATE TABLE contacts (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100),
    phone VARCHAR(20),
    company VARCHAR(100),
    category VARCHAR(50),
    created_at TIMESTAMP DEFAULT NOW()
);
```

---

### **3. Task Manager (Project Management Mini)**
**Fitur:**
- Create tasks dengan deadline
- Mark as complete/incomplete
- Priority level (high, medium, low)
- Filter by status & priority
- Due date reminders

**Database:**
```sql
CREATE TABLE tasks (
    id SERIAL PRIMARY KEY,
    title VARCHAR(255),
    description TEXT,
    status VARCHAR(20),
    priority VARCHAR(20),
    due_date DATE,
    completed BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT NOW()
);
```

---

### **4. Product Inventory (Stock Management)**
**Fitur:**
- Add products (name, price, stock, category)
- Update stock
- Low stock alert (<10)
- Search & filter products
- Total inventory value

**Database:**
```sql
CREATE TABLE products (
    id SERIAL PRIMARY KEY,
    name VARCHAR(200),
    price NUMERIC(10,2),
    stock INTEGER,
    category VARCHAR(50),
    description TEXT,
    created_at TIMESTAMP DEFAULT NOW()
);
```
