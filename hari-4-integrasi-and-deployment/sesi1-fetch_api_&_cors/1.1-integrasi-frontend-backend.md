## ğŸš¦ Masalah Umum: CORS Error

Saat Frontend (misal: port 5500) mencoba memanggil Backend (misal: port 3000), browser akan memblokirnya demi keamanan. Ini disebut **CORS Error**.

**Solusinya (di Backend):**
Kita harus memasang "pintu keluar masuk" agar Frontend diizinkan memanggil Backend.

```javascript
const cors = require('cors');
app.use(cors()); // Mengizinkan semua akses
```
