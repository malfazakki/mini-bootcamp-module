# Materi 1.2: Cara Mengambil Data (Fetch API)

## ðŸ“¡ Apa itu Fetch?

**Fetch** adalah perintah di JavaScript untuk mengambil data dari internet atau server kita sendiri.

---

## ðŸ“ Contoh Cara Mengambil Data (GET)

Gunakan kode ini di file `script.js` kamu:

```javascript
// 1. Panggil alamatnya
fetch("http://localhost:3000/berita")
  .then(respon => respon.json()) // 2. Ubah kiriman jadi format JSON
  .then(data => {
      // 3. Gunakan datanya (tampilkan ke layar)
      console.log(data);
  });
```

---

## ðŸ“ Contoh Cara Mengirim Data (POST)

```javascript
fetch("http://localhost:3000/berita", {
  method: "POST", // Kita bilang: "Saya mau ngirim!"
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ judul: "Berita Baru dari Web" })
});
```

---

## ðŸ’¡ Tips Penting
- **Check Status**: Jika hasil tidak muncul, cek di **F12 -> Network**.
- **JSON.stringify**: Gunakan ini saat mengirim data agar formatnya dimengerti server.
