# Full Stack Web Development Crash Course (4 Hari) - Mini Bootcamp Rumah IT

Selamat datang di modul pembelajaran intensif **Full Stack Web Development**. Kurikulum ini dirancang untuk membimbing pemula dari nol hingga berhasil men-deploy aplikasi web pertama mereka hanya dalam waktu 4 hari.

---

## Tujuan Pembelajaran
Setelah menyelesaikan 4 hari ini, peserta diharapkan mampu:
1.  Memahami cara kerja website (Client-Server Architecture).
2.  Membangun tampilan website interaktif (Frontend).
3.  Membuat server dan mengelola database (Backend).
4.  Menghubungkan Frontend & Backend serta meng-online-kan aplikasi (Deployment).

---

## Persiapan Tools (Wajib Install)
Sebelum memulai, pastikan komputer Anda sudah terpasang:
1.  **Code Editor:** [Visual Studio Code](https://code.visualstudio.com/)
2.  **Browser:** Google Chrome / Microsoft Edge
3.  **Runtime:** [Node.js (LTS Version)](https://nodejs.org/)
4.  **Database:** [PostgreSQL](https://www.postgresql.org/) & [PgAdmin](https://www.pgadmin.org/)
5.  **API Tester:** [Postman](https://www.postman.com/) (Opsional tapi disarankan)

---

## Jadwal & Kurikulum

### [Hari 1: Fundamental Web & Programming](./hari-1-fundamental-web-and-prgramming/)
*Fokus: Membangun fondasi struktur dan logika.*
*   **Sesi 1:** HTML - Struktur Tulang Punggung Website.
*   **Sesi 2:** CSS - Mempercantik Tampilan & Layouting (Flexbox).
*   **Sesi 3:** JavaScript Basic - Logika Pemrograman (Variabel, Fungsi, Loop).

### [Hari 2: Frontend Web Interaktif](./hari-2-frontend-web-interaktif/)
*Fokus: Membuat website yang "hidup" dan bisa merespon user.*
*   **Sesi 1:** DOM Manipulation - Mengubah HTML lewat JavaScript.
*   **Sesi 2:** Event Handling & Form Validation.
*   **Sesi 3:** **Mini Project:** Membuat Aplikasi "Expense Tracker" (Pencatat Keuangan) tanpa Database.

### [Hari 3: Backend & Database](./hari-3-backend-and-database/)
*Fokus: Membangun otak aplikasi dan penyimpanan data permanen.*
*   **Sesi 1:** Pengenalan Node.js & Express - Membuat Server API.
*   **Sesi 2:** Database PostgreSQL - Membuat Tabel & Relasi.
*   **Sesi 3:** CRUD API - Membuat fungsi Create, Read, Update, Delete data ke Database.

### [Hari 4: Integrasi & Deployment](./hari-4-integrasi-and-deployment/)
*Fokus: Menyatukan kepingan puzzle dan peluncuran.*
*   **Sesi 1:** Integrasi Frontend & Backend (Fetch API & CORS).
*   **Sesi 2:** Final Polish - Memperbaiki Bug & UI/UX.
*   **Sesi 3:** Deployment - Meng-online-kan Frontend (Vercel/Netlify) & Backend (Render/Railway).

---

## Tips Sukses Mengikuti Modul Ini
1.  **Jangan Cuma Copy-Paste:** Ketik ulang kodenya agar jari Anda terbiasa ("Muscle Memory").
2.  **Eksperimen:** Setelah kode jalan, coba ubah sedikit (misal ganti warna, ganti teks) untuk melihat efeknya.
3.  **Google is Your Friend:** Jika error, copy pesan error-nya ke Google. Kemampuan *debugging* adalah skill #1 programmer.
4.  **Istirahat:** Materi ini padat. Gunakan teknik Podomoro (25 menit fokus, 5 menit istirahat) agar otak tidak *overheat*.
