**Node.js** adalah alat yang membuat kita bisa menjalankan JavaScript di komputer (bukan hanya di browser). Dengan Node.js, kita bisa membuat server.

**Express.js** adalah "asisten" yang membuat proses pembuatan server jadi sangat mudah dan cepat.

---

## ğŸ› ï¸ Langkah Membuat Server (Minimalis)

Cukup simpan kode ini di file bernama `server.js`:

```javascript
const express = require("express"); // Ambil asisten Express
const app = express();              // Mulai aplikasi
const port = 3000;                  // Tentukan nomor pintu (port)

// Jika ada yang buka pintu utama "/", kirim tulisan "Halo!"
app.get("/", (req, res) => {
    res.send("Halo! Ini adalah server buatan saya.");
});

// Jalankan server
app.listen(port, () => {
    console.log("Server sudah jalan di http://localhost:" + port);
});
```
