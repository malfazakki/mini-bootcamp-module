## ğŸŒ Apa itu Web Server?

**Web Server** adalah software yang:
1. Menerima **HTTP Request** dari client (browser)
2. Memproses request (ambil data, jalankan logika)
3. Mengirim **HTTP Response** kembali ke client

**Contoh Web Server:**
- **Apache** (paling populer)
- **Nginx** (cepat & ringan)
- **Node.js** (yang akan kita pakai)

## ğŸ”„ Request-Response Cycle

Mari kita lihat step-by-step saat user buka website:

### **1. User Buka Website**
```
User ketik di browser: https://mywebsite.com/products
```

### **2. Browser Kirim Request**
```http
GET /products HTTP/1.1
Host: mywebsite.com
User-Agent: Chrome/120.0
Accept: text/html
```

### **3. Server Terima & Proses**
```javascript
// Server menerima request
app.get('/products', async (req, res) => {
    // Ambil data produk dari database
    const products = await db.query('SELECT * FROM products');
    
    // Kirim response
    res.json(products);
});
```

### **4. Server Kirim Response**
```http
HTTP/1.1 200 OK
Content-Type: application/json

[
    { "id": 1, "name": "Sepatu Nike", "price": 500000 },
    { "id": 2, "name": "Tas Adidas", "price": 300000 }
]
```

### **5. Browser Render**
Browser menerima data dan menampilkannya ke user.
