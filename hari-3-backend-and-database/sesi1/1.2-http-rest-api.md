# Materi 1.2: Cara Website Mengobrol (HTTP)

## ðŸ“¡ Protokol HTTP

Website menggunakan "bahasa" yang disebut **HTTP** untuk mengirim pesan antara pelayan (Frontend) dan dapur (Backend).

Ada 2 jenis pesan yang peling penting:
1.  **GET**: Digunakan untuk **mengambil** data. (Misal: melihat daftar berita).
2.  **POST**: Digunakan untuk **mengirim** data. (Misal: mengirim formulir pendaftaran).

---

## ðŸ’» Contoh Kode 'Ngobrol'

Inilah cara Frontend meminta data ke Backend menggunakan JavaScript:

```javascript
// Meminta daftar berita dari server
fetch("http://localhost:3000/berita")
  .then(response => response.json()) // Ubah kiriman jadi format JSON
  .then(data => {
      console.log("Datanya adalah:", data);
  });
```

---

## ðŸš¦ Kode Status (Status Codes)

Setelah Backend memproses pesan, dia akan kasih kode angka:
-   **200 (OK)**: Berhasil!
-   **404 (Not Found)**: Maaf, halaman tidak ada.
-   **500 (Server Error)**: Maaf, dapurnya lagi rusak/error.

---

## ðŸ¥— Apa itu JSON?

Data yang dikirim biasanya dibungkus dalam format bernama **JSON**. Bentuknya seperti daftar belanja:

```json
{
  "nama": "Budi",
  "status": "Mahasiswa",
  "hobi": ["Ngoding", "Gaming"]
}
```
