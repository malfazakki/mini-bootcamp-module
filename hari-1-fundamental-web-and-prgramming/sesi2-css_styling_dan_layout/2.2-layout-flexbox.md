# Materi 2.2: Mengatur Posisi dengan Flexbox

## ğŸ§© Apa itu Flexbox?

**Flexbox** adalah "Kotak Ajaib" di CSS. Gunanya untuk mengatur posisi elemen agar bisa berjejer ke samping atau ke bawah dengan sangat mudah.

Dulu, mengatur posisi sangat sulit. Sekarang dengan Flexbox, cukup satu-dua baris kode, semua langsung rapi.

---

## ğŸ—ï¸ Cara Menggunakan Flexbox

Flexbox bekerja dengan konsep **Wadah (Container)** dan **Isi (Items)**. Kita cukup memberi perintah pada si Wadah.

```html
<div class="wadah">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
</div>
```

```css
.wadah {
    display: flex;  /* â† Wajib ada untuk mengaktifkan Flexbox */
}
```

---

## ğŸ“ 4 Perintah Utama Flexbox

Hanya dengan 4 perintah ini, kamu sudah bisa mengatur layout website:

### 1. `justify-content` (Geser Horizontal / Kiri-Kanan)
*   `justify-content: center;` (Semua di tengah)
*   `justify-content: space-between;` (Mencar ke ujung kiri dan kanan)

### 2. `align-items` (Geser Vertikal / Atas-Bawah)
*   `align-items: center;` (Tengah-tengah secara vertikal)

### 3. `flex-direction` (Arah Jejeran)
*   `flex-direction: row;` (Default: Jejer ke samping)
*   `flex-direction: column;` (Jejer ke bawah)

### 4. `gap` (Jarak Antar Item)
Memberi jarak antar kotak tanpa perlu ribet dengan margin.
*   `gap: 20px;`

Ini adalah teknik yang paling sering dicari oleh pemula.

```css
body {
    display: flex;
    justify-content: center; /* Tengah kiri-kanan */
    align-items: center;     /* Tengah atas-bawah */
    height: 100vh;           /* Tinggi layar penuh */
}
```
