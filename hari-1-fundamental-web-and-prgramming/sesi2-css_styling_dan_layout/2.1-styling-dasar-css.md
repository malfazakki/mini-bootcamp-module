# Materi 2.1: Menghias Website dengan CSS

## ğŸ¨ Apa itu CSS?

Jika HTML adalah **rangka**, maka **CSS** (Cascading Style Sheets) adalah **cat dan dekorasinya**. CSS bertugas membuat website kita terlihat cantik dengan warna, ukuran font, dan jarak yang pas.

Tanpa CSS, website hanya akan berisi teks hitam-putih yang membosankan.

---

## ğŸ”— Menghubungkan CSS ke HTML

Cara terbaik adalah menggunakan file terpisah agar kode kita rapi.

1.  Buat file baru bernama `style.css`.
2.  Di dalam file HTML (bagian `<head>`), tambahkan baris ini:
    ```html
    <link rel="stylesheet" href="style.css">
    ```

---

## ğŸ“ Aturan Menulis CSS

Sintaks CSS sangat sederhana:
```css
selector {
    property: value;
}
```

*   **Selector**: Nama elemen yang mau dihias (misal: `h1`, `p`, atau nama class).
*   **Property**: Apa yang mau diubah (misal: warna, ukuran).
*   **Value**: Nilai barunya (misal: biru, 20px).

---

## ğŸ¯ Selector yang Paling Sering Digunakan

### 1. Tag Selector (Semua h1, semua p)
```css
h1 {
    color: blue;
}
```

### 2. Class Selector (Hanya yang kita tandai)
Di HTML, kita beri tanda: `<p class="merah">TEKS</p>`.
Di CSS, gunakan tanda **titik (.)**:
```css
.merah {
    color: red;
}
```

---

## ğŸ“¦ Properti Dasar yang Wajib Diketahui

### 1. Warna & Font
*   `color`: Mengubah warna teks.
*   `background-color`: Mengubah warna latar.
*   `font-size`: Mengubah ukuran teks (contoh: `20px`).
*   `font-family`: Mengubah jenis tulisan (contoh: `Arial`).

### 2. Isi & Jarak (Box Model)
Bayangkan setiap elemen adalah kotak:
*   **Padding**: Jarak antara teks dengan garis tepi (jarak di dalam).
*   **Margin**: Jarak antara kotak satu dengan kotak lainnya (jarak di luar).
*   **Border**: Garis tepi/bingkai kotak.
*   **Border-radius**: Membuat sudut kotak jadi melengkung (tidak tajam).

```css
.kotak {
    background-color: lightgrey;
    padding: 20px;
    margin: 10px;
    border: 2px solid black;
    border-radius: 10px;
}
```
