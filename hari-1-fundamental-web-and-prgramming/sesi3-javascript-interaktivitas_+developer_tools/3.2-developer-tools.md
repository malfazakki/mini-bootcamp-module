## ğŸ”§ 2. Menyimpan Kode ke GitHub

**GitHub** adalah tempat kita menyimpan kode secara online agar tidak hilang dan bisa dilihat orang lain.

### **Workflow 5 Langkah Ajaib:**

1.  **git init**: Memberitahu komputer "Folder ini akan dipantau oleh Git".
2.  **git status**: Mengecek file apa saja yang baru atau berubah.
3.  **git add .**: Menandai semua file untuk siap disimpan.
4.  **git commit -m "pesan"**: Menyimpan secara permanen dengan catatan kecil.
5.  **git push**: Mengirim file dari komputermu ke internet (GitHub).
