## âš¡ Apa itu JavaScript?

Jika HTML itu **rangka** dan CSS itu **cat**, maka **JavaScript** adalah **otot**. JavaScript bertugas membuat website bisa "berpikir" dan melakukan aksi.

Contoh: Menghitung angka, menampilkan pesan selamat datang, atau mengubah warna layar saat tombol diklik.

## ðŸ“¦ Dasar-Dasar yang Penting

### 1. Variabel (Wadah Penyimpan)
Gunakan `let` untuk data yang bisa berubah, dan `const` untuk data yang tetap.
```javascript
let nama = "Budi";
const tahunLahir = 2000;
```

### 2. Fungsi (Blok Perintah)
Fungsi adalah kumpulan perintah yang baru jalan kalau "dipanggil".
```javascript
function sapaSaya() {
    alert("Halo, selamat datang!");
}
```

## ðŸŽ¯ Mengolah Input dari User

Ini adalah bagian paling seru! Kita akan mengambil apa yang diketik user dan menampilkannya.

### **Di HTML:**
```html
<input type="text" id="namaUser" placeholder="Ketik namamu">
<button onclick="prosesData()">Klik Saya</button>
<h2 id="hasil"></h2>
```

### **Di JavaScript (`script.js`):**
```javascript
function prosesData() {
    // 1. Ambil apa yang diketik di input
    let inputan = document.getElementById("namaUser").value;

    // 2. Tampilkan ke dalam tag H2 (id="hasil")
    document.getElementById("hasil").innerHTML = "Halo " + inputan + "!";
}
```
