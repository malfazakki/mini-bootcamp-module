# Materi 3.1: Dasar Pemrograman JavaScript

## üíª Apa itu JavaScript?

**JavaScript** (disingkat **JS**) adalah bahasa pemrograman yang membuat halaman web menjadi **interaktif** dan **dinamis**.

**üí° Analogi Rumah:**
- **HTML** = Rangka rumah (struktur)
- **CSS** = Cat & dekorasi (tampilan)
- **JavaScript** = Listrik, AC, alarm (fungsi & interaksi)

**Contoh Penggunaan JavaScript:**
- Validasi form sebelum dikirim
- Animasi saat scroll
- Popup/Modal yang muncul saat klik tombol
- Menampilkan data dari server tanpa refresh halaman
- Calculator, game, dan aplikasi web interaktif

---

1. **Di Browser** (Client-Side): 
   - Chrome, Firefox, Safari, Edge
   - Dijalankan oleh mesin JavaScript di browser

2. **Di Server** (Server-Side):
   - **Node.js** (akan kita pelajari di Hari 3)

Untuk sekarang, kita fokus ke JavaScript di browser dulu.

## üì¶ Variabel: Menyimpan Data

Variabel adalah "wadah" untuk menyimpan data.

### **3 Cara Deklarasi Variabel:**

#### 1Ô∏è‚É£ **`let`** (Modern, bisa diubah)
```javascript
let nama = "Budi";
console.log(nama);  // Output: Budi

nama = "Ani";  // Bisa diubah
console.log(nama);  // Output: Ani
```

#### 2Ô∏è‚É£ **`const`** (Modern, TIDAK bisa diubah)
```javascript
const umur = 25;
console.log(umur);  // Output: 25

// umur = 30;  // ‚ùå ERROR! const tidak bisa diubah
```

**Kapan pakai `const`?**
- Untuk nilai yang tidak akan berubah (contoh: PI = 3.14)
- Untuk object dan array (isinya bisa berubah, tapi referensinya tetap)

#### 3Ô∏è‚É£ **`var`** (Lama, hindari di kode modern)
```javascript
var kota = "Jakarta";
```

**üí° Best Practice:**
- Gunakan `const` secara default
- Gunakan `let` jika nilai akan berubah
- Hindari `var`

## ‚ûï Operator

### **Operator Aritmatika:**
```javascript
let x = 10;
let y = 3;

console.log(x + y);  // 13
console.log(x - y);  // 7
console.log(x * y);  // 30
console.log(x / y);  // 3.333...
console.log(x % y);  // 1 (sisa bagi)

// Increment & Decrement
x++;  // x = x + 1 (sekarang x = 11)
y--;  // y = y - 1 (sekarang y = 2)
```

### **Operator Perbandingan:**
```javascript
console.log(5 == "5");   // true (nilai sama, tipe tidak dicek)
console.log(5 === "5");  // false (nilai DAN tipe harus sama) ‚úÖ GUNAKAN INI
console.log(5 != "5");   // false
console.log(5 !== "5");  // true ‚úÖ GUNAKAN INI

console.log(10 > 5);   // true
console.log(10 < 5);   // false
console.log(10 >= 10); // true
console.log(5 <= 3);   // false
```

**üí° Best Practice:** Gunakan `===` dan `!==` (strict equality).

### **Operator Logika:**
```javascript
let umur = 20;
let punyaKTP = true;

// AND (&&) - Semua harus true
console.log(umur >= 17 && punyaKTP);  // true

// OR (||) - Salah satu true
console.log(umur >= 17 || punyaKTP);  // true

// NOT (!) - Membalik nilai
console.log(!punyaKTP);  // false
```

## üîÄ Kondisional (If-Else)

### **If-Else:**
```javascript

if (nilai >= 90) {
    console.log("Grade: A");
} else if (nilai >= 80) {
    console.log("Grade: B");
} else if (nilai >= 70) {
```

### **Ternary Operator (Shorthand):**
```javascript

console.log(status);  // Dewasa
```

### **Switch Case:**
```javascript
let hari = "Senin";

switch (hari) {
    case "Senin":
        console.log("Awal pekan");
        break;
    case "Jumat":
        console.log("Akhir pekan kerja");
        break;
    case "Sabtu":
    case "Minggu":
        console.log("Weekend!");
        break;
    default:
        console.log("Hari biasa");
}
```

## üéØ Fungsi (Functions)

Fungsi adalah blok kode yang bisa dipanggil berulang kali.

### **Function Declaration:**
```javascript
function sapa(nama) {
    return "Halo, " + nama + "!";
}

console.log(sapa("Budi"));  // Halo, Budi!
console.log(sapa("Ani"));   // Halo, Ani!
```

### **Arrow Function (Modern):**
```javascript
const tambah = (a, b) => {
    return a + b;
};

// Atau lebih singkat (jika 1 baris):
const kali = (a, b) => a * b;

console.log(tambah(5, 3));  // 8
console.log(kali(4, 2));    // 8
```

### **Function dengan Default Parameter:**
```javascript
function sapa(nama = "Tamu") {
    return `Halo, ${nama}!`;
}

console.log(sapa());        // Halo, Tamu!
console.log(sapa("Budi"));  // Halo, Budi!
```

## üéÆ Contoh Project: Calculator Sederhana

```javascript
// Fungsi untuk operasi matematika
function tambah(a, b) {
    return a + b;
}

function kurang(a, b) {
    return a - b;
}

function kali(a, b) {
    return a * b;
}

function bagi(a, b) {
    if (b === 0) {
        return "Error: Tidak bisa bagi 0";
    }
    return a / b;
}

// Fungsi utama calculator
function calculator(angka1, operator, angka2) {
    switch (operator) {
        case '+':
            return tambah(angka1, angka2);
        case '-':
            return kurang(angka1, angka2);
        case '*':
            return kali(angka1, angka2);
        case '/':
            return bagi(angka1, angka2);
        default:
            return "Operator tidak valid";
    }
}

// Testing
console.log(calculator(10, '+', 5));  // 15
console.log(calculator(10, '-', 5));  // 5
console.log(calculator(10, '*', 5));  // 50
console.log(calculator(10, '/', 5));  // 2
console.log(calculator(10, '/', 0));  // Error: Tidak bisa bagi 0
```

## üõçÔ∏è Contoh Project: Hitung Total Belanja

```javascript
// Data produk (array of objects)
const keranjang = [
    { nama: "Baju", harga: 100000, jumlah: 2 },
    { nama: "Celana", harga: 150000, jumlah: 1 },
    { nama: "Sepatu", harga: 300000, jumlah: 1 }
];

// Fungsi hitung subtotal per item
function hitungSubtotal(item) {
    return item.harga * item.jumlah;
}

// Fungsi hitung total semua
function hitungTotal(keranjang) {
    let total = 0;
    keranjang.forEach(item => {
        total += hitungSubtotal(item);
    });
    return total;
}

// Fungsi terapkan diskon
function terapkanDiskon(total, persenDiskon) {
    const diskon = total * (persenDiskon / 100);
    return total - diskon;
}

// Jalankan
const totalSebelumDiskon = hitungTotal(keranjang);
console.log("Total Sebelum Diskon: Rp " + totalSebelumDiskon);

const totalSetelahDiskon = terapkanDiskon(totalSebelumDiskon, 10);
console.log("Total Setelah Diskon 10%: Rp " + totalSetelahDiskon);

// Output:
// Total Sebelum Diskon: Rp 550000
// Total Setelah Diskon 10%: Rp 495000
```

### **Latihan 1: FizzBuzz**
Buat program yang mencetak angka 1-30, tapi:
- Jika angka habis dibagi 3, cetak "Fizz"
- Jika angka habis dibagi 5, cetak "Buzz"  
- Jika habis dibagi 3 DAN 5, cetak "FizzBuzz"

```javascript
for (let i = 1; i <= 30; i++) {
    if (i % 3 === 0 && i % 5 === 0) {
        console.log("FizzBuzz");
    } else if (i % 3 === 0) {
        console.log("Fizz");
    } else if (i % 5 === 0) {
        console.log("Buzz");
    } else {
        console.log(i);
    }
}
```

### **Latihan 2: Palindrome Checker**
Cek apakah sebuah kata adalah palindrome (sama jika dibaca terbalik).

```javascript
function isPalindrome(kata) {
    const terbalik = kata.split('').reverse().join('');
    return kata.toLowerCase() === terbalik.toLowerCase();
}

console.log(isPalindrome("katak"));  // true
console.log(isPalindrome("malam"));  // true
console.log(isPalindrome("halo"));   // false
```

### **Latihan 3: Konversi Suhu**
Buat fungsi untuk konversi Celsius ke Fahrenheit.

```javascript
function celsiusToFahrenheit(celsius) {
    return (celsius * 9/5) + 32;
}

console.log(celsiusToFahrenheit(0));   // 32
console.log(celsiusToFahrenheit(100)); // 212
console.log(celsiusToFahrenheit(37));  // 98.6
```
