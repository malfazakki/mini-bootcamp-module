#### **2ï¸âƒ£ Console Tab** (JavaScript REPL)

**Kegunaan:**
- Menjalankan JavaScript langsung
- Melihat `console.log()` output

**Contoh:**

#### **3ï¸âƒ£ Network Tab**

**Info yang bisa dilihat:**

#### **4ï¸âƒ£ Sources Tab**

## ğŸ”§ 3. Git & GitHub

**Git** = Software version control (tracking perubahan kode)
**GitHub** = Platform cloud untuk hosting repository Git

### **ğŸ’¡ Analogi:**
Git seperti "Google Docs History" untuk kode:
- Bisa lihat semua perubahan
- Bisa kembali ke versi sebelumnya
- Bisa kolaborasi dengan tim

---

### **ğŸ“¥ Install Git:**

**Windows:**
1. Download dari: https://git-scm.com/
2. Install dengan setting default
3. Cek instalasi: Buka CMD â†’ ketik `git --version`

**Hasil:**
```
git version 2.43.0
```

---

### **ğŸ“ Konfigurasi Awal Git:**

```bash
# Set username
git config --global user.name "Nama Anda"

# Set email (pakai email GitHub)
git config --global user.email "email@example.com"

# Cek konfigurasi
git config --list
```

---

### **ğŸ“¦ Workflow Git Dasar:**

#### **1. Inisialisasi Repository:**
```bash
# Masuk ke folder project
cd C:\Users\programmer\Documents\my-project

# Inisialisasi Git
git init

# Akan muncul folder .git (hidden)
```

#### **2. Cek Status:**
```bash
git status
```

**Output:**
```
On branch main
Untracked files:
  index.html
  style.css
```

#### **3. Staging (Tambahkan ke Tracking):**
```bash
# Tambah file specific
git add index.html

# Atau tambah semua file
git add .
```

#### **4. Commit (Simpan Snapshot):**
```bash
git commit -m "Initial commit: Add index and style"
```

**ğŸ’¡ Tips Commit Message:**
- Gunakan bahasa Inggris
- Singkat & deskriptif
- Contoh bagus: "Add login form validation"
- Contoh buruk: "update" atau "fix"

---

### **â˜ï¸ Push ke GitHub:**

#### **1. Buat Repository di GitHub:**
1. Login ke https://github.com
2. Klik tombol **New Repository**
3. Isi nama repo (misal: `my-first-website`)
4. Klik **Create Repository**

#### **2. Hubungkan Local ke GitHub:**
```bash
# Tambah remote origin
git remote add origin https://github.com/username/my-first-website.git

# Cek remote
git remote -v
```

#### **3. Push Code:**
```bash
# Push ke branch main
git push -u origin main
```

**Selesai!** Kode Anda sekarang ada di cloud dan bisa diakses siapa saja! ğŸ‰
