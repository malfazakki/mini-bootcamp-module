## ğŸ”§ 2. Git & GitHub

**Git** = Software untuk mencatat setiap perubahan kode Anda (seperti Save Point di Game).
**GitHub** = Website untuk menyimpan folder (repository) Git Anda di internet (Cloud).

### **ï¿½ Workflow Dasar Git:**

1. **git init**: Inisialisasi folder menjadi folder Git.
2. **git status**: Cek file apa saja yang berubah.
3. **git add .**: Menandai file untuk disimpan.
4. **git commit -m "pesan"**: Menyimpan perubahan dengan catatan.
5. **git push**: Mengirim hasil simpanan ke GitHub.

---

### **ï¿½ Konfigurasi Awal (Hanya Sekali):**

```bash

git config --global user.email "email@anda.com"
```
