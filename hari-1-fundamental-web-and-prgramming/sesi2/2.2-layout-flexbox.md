# Materi 2.2: Layout menggunakan Flexbox

## ğŸ§© Apa itu Flexbox?

**Flexbox** (Flexible Box Layout) adalah modul CSS3 yang memudahkan kita mengatur tata letak (layout) elemen di halaman web, terutama untuk membuat elemen sejajar secara horizontal atau vertikal.

**ğŸ’¡ Analogi:**
Bayangkan Anda mengatur barang-barang di dalam sebuah kotak:
- Flexbox = Kotak ajaib yang bisa mengatur posisi barang secara otomatis
- Anda bisa atur: "Semua barang di tengah", "Barang rapat kiri", "Jarak sama rata", dll.

**Sebelum Flexbox:**
Kita pakai `float`, `table`, atau trik-trik CSS yang rumit dan sering error.

**Dengan Flexbox:**
Semua jadi mudah! ğŸ‰

## ğŸ—ï¸ Konsep Dasar Flexbox

Flexbox bekerja dengan 2 komponen:
1. **Flex Container** (Parent/Kotak luar)
2. **Flex Items** (Children/Isi kotak)

```html
<div class="container">  <!-- Flex Container -->
    <div class="item">Item 1</div>  <!-- Flex Item -->
    <div class="item">Item 2</div>  <!-- Flex Item -->
    <div class="item">Item 3</div>  <!-- Flex Item -->
</div>
```

```css
.container {
    display: flex;  /* â† Aktifkan Flexbox! */
}
```

### 2ï¸âƒ£ **`flex-direction`**


## ğŸ§± Properti untuk Flex Items (Children)

### 1ï¸âƒ£ **`flex-grow`**
Menentukan seberapa besar item akan "tumbuh" untuk mengisi ruang kosong.

```css
.item1 {
    flex-grow: 1;  /* Akan mengambil 1 bagian ruang kosong */
}

.item2 {
    flex-grow: 2;  /* Akan mengambil 2 bagian (2x lebih besar dari item1) */
}
```

**Contoh:**
```html
<div class="container">
    <div class="item" style="flex-grow: 1;">Item 1</div>
    <div class="item" style="flex-grow: 2;">Item 2 (Larger)</div>
    <div class="item" style="flex-grow: 1;">Item 3</div>
</div>
```

Item 2 akan **2x lebih lebar** dari Item 1 dan 3.

---

### 2ï¸âƒ£ **`flex-shrink`**
Menentukan seberapa besar item akan "mengecil" saat ruang tidak cukup.

```css
.item {
    flex-shrink: 1;  /* Default, akan mengecil */
    /* flex-shrink: 0; */  /* Tidak akan mengecil */
}
```

---

### 3ï¸âƒ£ **`flex-basis`**
Ukuran awal item sebelum space didistribusikan.

```css
.item {
    flex-basis: 200px;  /* Ukuran awal 200px */
}
```

---

### 4ï¸âƒ£ **Shorthand: `flex`**
Menggabungkan `flex-grow`, `flex-shrink`, dan `flex-basis`.

```css
.item {
    flex: 1;  /* Sama dengan: flex-grow: 1, flex-shrink: 1, flex-basis: 0 */
    /* flex: 2; */       /* Akan 2x lebih besar */
    /* flex: 0 0 200px; */  /* Tidak grow/shrink, basis 200px */
}
```

## ğŸ“± Flexbox untuk Responsive Design

Kombinasikan dengan Media Queries:

```css
.card-container {
    display: flex;
    flex-wrap: wrap;  /* Baris baru jika tidak cukup ruang */
    gap: 20px;
}

.card {
    flex: 1 1 300px;  /* Minimal 300px, bisa grow */
}

/* Untuk layar kecil (mobile) */
@media screen and (max-width: 768px) {
    .card-container {
        flex-direction: column;  /* Jadi vertikal */
    }
    
    .navbar {
        flex-direction: column;
        gap: 10px;
    }
}
```

**Tugas 1: Menu Navigasi**

## ğŸ“š Cheat Sheet Flexbox

```css

flex-direction: row | column;

```
