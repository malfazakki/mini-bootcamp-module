## ğŸ“ Sintaks Dasar CSS

Semua aturan CSS mengikuti pola ini:

```css
selector {
    property: value;
    property: value;
}
```

**Contoh:**
```css
h1 {
    color: blue;
    font-size: 32px;
    text-align: center;
}
```

**Penjelasan:**
- `h1` â†’ **Selector** (elemen yang akan di-styling)
- `color`, `font-size`, `text-align` â†’ **Property** (aspek yang diubah)
- `blue`, `32px`, `center` â†’ **Value** (nilai yang diberikan)

---

## ğŸ¯ CSS Selectors (Cara Memilih Elemen)

### 1ï¸âƒ£ **Tag Selector**
Memilih semua elemen dengan tag tertentu.

```css
p {
    color: blue;
}
```
**Artinya:** Semua tag `<p>` akan berwarna biru.

---

### 2ï¸âƒ£ **Class Selector**
Memilih elemen berdasarkan class. Gunakan **titik (.)** di depan nama class.

**HTML:**
```html
<p class="highlight">Teks ini special</p>
<p>Teks biasa</p>
<p class="highlight">Teks ini juga special</p>
```

**CSS:**
```css
.highlight {
    background-color: yellow;
    font-weight: bold;
}
```

**Hasilnya:** Hanya paragraf dengan `class="highlight"` yang akan memiliki background kuning.

**ğŸ’¡ Tips:** Class bisa dipakai berulang kali di banyak elemen.

---

### 3ï¸âƒ£ **ID Selector**
Memilih elemen berdasarkan ID. Gunakan **hashtag (#)** di depan nama ID.

**HTML:**
```html
<div id="header">Ini adalah header</div>
```

**CSS:**
```css
#header {
    background-color: navy;
    color: white;
    padding: 20px;
}
```

**âš ï¸ Penting:** ID harus unik! Hanya boleh ada 1 elemen dengan ID yang sama di satu halaman.

---

### ğŸ“Š Tabel Perbandingan Selector

| **Selector** | **Sintaks** | **Dipakai Untuk** | **Contoh** |
|--------------|-------------|-------------------|------------|
| Tag | `p { }` | Semua elemen `<p>` | Styling semua paragraf |
| Class | `.nama-class { }` | Elemen dengan class tertentu | Tombol dengan style sama |
| ID | `#nama-id { }` | 1 elemen unik | Header utama |

## ğŸ¨ Properti CSS yang Sering Dipakai

### **1. Warna (Colors)**

```css

    color: red;                    /* Warna teks */

```

**Cara Menulis Warna:**
- **Nama warna:** `red`, `blue`, `green`
- **HEX:** `#FF5733`, `#000000`
- **RGB:** `rgb(255, 87, 51)`
- **RGBA:** `rgba(255, 87, 51, 0.5)` â† Ada transparansi

---

### **2. Text Styling**

```css
p {
    font-size: 18px;              /* Ukuran huruf */
    font-family: Arial, sans-serif; /* Jenis font */
    font-weight: bold;            /* Ketebalan (normal, bold) */
    text-align: center;           /* Rata tengah */
    text-decoration: underline;   /* Garis bawah */
    line-height: 1.6;             /* Jarak antar baris */
    letter-spacing: 2px;          /* Jarak antar huruf */
}
```
