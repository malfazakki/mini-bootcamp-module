## ğŸŒ Bagaimana Web Bekerja? (Untuk Pemula)

Ketika Anda mengetik **www.google.com** di browser dan tekan Enter, inilah yang terjadi:

### **1. Request (Permintaan)**
Browser Anda mengirim permintaan ke server Google:
```
"Halo Google, saya mau lihat halaman utama"
```

### **2. Server Processing (Pemrosesan di Server)**
Server Google menerima permintaan Anda dan memproses:
```
- "OK, pengguna mau halaman utama"
- "Ambil file HTML, CSS, JavaScript"
- "Cek apakah pengguna sudah login"
- "Siapkan data yang diperlukan"
```

### **3. Response (Balasan)**
Server mengirimkan file-file ke browser Anda:
```
ğŸ“„ index.html (halaman utama)
ğŸ¨ style.css (tampilan)
âš™ï¸ script.js (fitur interaktif)
ğŸ“Š data.json (data pengguna)
```

### **4. Rendering (Tampilan di Browser)**
Browser menerima semua file dan menampilkan halaman Google yang Anda kenal.
