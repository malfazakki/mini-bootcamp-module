**Code Editor** adalah aplikasi tempat kita menulis kode program. Ini berbeda dengan Microsoft Word. Code editor didesain khusus agar kode kita mudah dibaca, rapi, dan bisa dijalankan oleh komputer.

Dalam bootcamp ini, kita akan menggunakan **Visual Studio Code (VS Code)** karena ringan, gratis, dan banyak digunakan oleh profesional.

## ğŸ“¦ Extension Penting (Live Server)

Extension adalah fitur tambahan untuk mempercanggih VS Code. Kita hanya butuh satu yang paling utama: **Live Server**.

**Fungsi**: Membuat website kita terbuka di browser secara otomatis. Saat kamu simpan (Save) kode, tampilan di browser akan langsung update tanpa perlu refresh manual.

**Cara Instal**:
1.  Klik ikon kotak di sebelah kiri VS Code (**Extensions**).
2.  Cari: "Live Server".
3.  Klik tombol **Install**.

## âŒ¨ï¸ Shortcut Dasar (Wajib Tahu)

Gunakan ini agar kerjamu makin cepat:
*   **Ctrl+S** (Windows) / **Cmd+S** (Mac): Simpan file (Lakukan setiap kali ada perubahan!)
*   **Ctrl+N** / **Cmd+N**: Buat file baru.
*   **Ctrl+/ ** / **Cmd+/**: Memberi komentar pada kode (agar tidak dibaca komputer).
