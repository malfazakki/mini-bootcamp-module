**Code Editor** adalah perangkat lunak utama yang digunakan programmer untuk menulis kode. Ibarat tukang kayu yang butuh meja kerja, programmer butuh Code Editor untuk menyusun aplikasi mereka.

Di masa kini, kita tidak hanya punya Code Editor biasa, tapi juga **IDE AI (Artificial Intelligence)** yang bisa membantu kita menulis kode lebih cepat!

Editor ini dibangun khusus dengan AI di jantungnya:
