## ğŸŒ Bagaimana Cara Kerja Website?

Bayangkan seperti memesan kopi di kafe:

1.  **Request (Pesan)**: Kamu (sebagai **Client/Browser**) mengetik alamat web dan menekan Enter. Itu seperti bilang ke pelayan: "Saya pesan halaman utama Google!"
2.  **Process (Masak)**: Server menerima pesanmu dan menyiapkan "bahan-bahannya" (file HTML/CSS).
3.  **Response (Sajikan)**: Server mengirim balik file-file tersebut ke browsermu.
4.  **Tampilan**: Browser menyusun file tersebut menjadi tampilan cantik yang kamu lihat.
