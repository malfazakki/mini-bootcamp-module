**Responsive** artinya website kita bisa berubah tampilan secara otomatis agar tetap rapi saat dibuka di HP, Tablet, atau Laptop.

## ï¿½ 1. Syarat Wajib: Viewport Tag

Agar website tidak terlihat sangat kecil di HP, kita **wajib** memasang kode ini di dalam `<head>`:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## ğŸ¨ 2. Media Query (Tombol Otomatis)

Media Query adalah cara kita bilang ke CSS: "Jika layar HP sempit, tolong ganti gayanya."

**Contoh: Mengubah Warna Background di HP**
```css
/* Gaya untuk Laptop (default) */
body {
    background-color: white;
}

/* Gaya khusus Layar di bawah 600px (HP) */
@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}
```

---

## ğŸ—ï¸ 3. Teknik "Susun Tumpuk"

Biasanya di laptop kita susun kotak ke samping (kiri-kanan). Di HP, kita buat kotak itu menumpuk (atas-bawah).

**Contoh Sederhana:**
```css
.kontainer {
    display: flex;
    flex-direction: row; /* Laptop: Ke samping */
}

@media (max-width: 600px) {
    .kontainer {
        flex-direction: column; /* HP: Menumpuk ke bawah */
    }
}
```
