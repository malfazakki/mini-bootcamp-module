## ðŸ“š 1. Struktur Data Kompleks

Di hari pertama kita hanya menggunakan teks (string) dan angka (number). Sekarang kita akan belajar menyimpan banyak data sekaligus.

### A. Array (Daftar Data)
Array adalah wadah untuk menyimpan banyak nilai dalam satu variabel. Bayangkan seperti rak buku yang bernomor.

```javascript
// Membuat Array
const buah = ["Apel", "Mangga", "Jeruk"];

// Mengakses Data (Indeks mulai dari 0)
console.log(buah[0]); // Output: Apel
console.log(buah[2]); // Output: Jeruk

// Menambah Data
buah.push("Pisang"); // Menambah ke urutan terakhir

// Mengetahui jumlah data
console.log(buah.length); // Output: 4
```

### B. Object (Kumpulan Informasi)
Object digunakan untuk mendeskripsikan suatu benda dengan properti-propertinya.

```javascript
// Membuat Object
const siswa = {
    nama: "Budi",
    umur: 17,
    kelas: "XII RPL",
    lulus: true
};

// Mengakses Data
console.log(siswa.nama);  // Output: Budi (Dot notation - Paling sering dipakai)
console.log(siswa["umur"]); // Output: 17 (Bracket notation)
```

## ðŸš€ 2. Fitur Modern JavaScript (ES6+)

Fitur ini membuat penulisan kode lebih ringkas, rapi, dan modern.

### A. Template Literals (Backticks)
Dulu kita menggabungkan teks dengan tanda `+`. Sekarang kita pakai backtick (`` ` ``) dan `${}`.

**Cara Lama:**
```javascript
let nama = "Budi";
console.log("Halo, nama saya " + nama + "!");
```

**Cara Modern (ES6):**
```javascript
const nama = "Budi";
console.log(`Halo, nama saya ${nama}!`); 
// Lebih bersih, tidak perlu pusing dengan tanda kutip dan plus
```

### B. Arrow Function
Penulisan fungsi yang lebih singkat.

**Cara Lama:**
```javascript
function sapa(nama) {
    return "Halo " + nama;
}
```

**Cara Modern (ES6):**
```javascript
const sapa = (nama) => {
    return `Halo ${nama}`;
};

// Atau jika hanya satu baris return, bisa lebih singkat lagi:
const sapaSingkat = (nama) => `Halo ${nama}`;
```

## ðŸ”„ 3. Looping dan Array Methods

Alih-alih menggunakan `for` loop manual yang panjang, JavaScript modern menyediakan metode bawaan yang "sakti" untuk mengolah data array.

Misalkan kita punya data nilai:
```javascript
const nilai = [60, 80, 90, 50, 75];
```

### A. .forEach() - Melakukan aksi untuk setiap item
Hanya menjalankan kode, tidak menghasilkan array baru.
```javascript
nilai.forEach((item) => {
    console.log(`Nilai siswa: ${item}`);
});
```

### B. .map() - Mengubah setiap item (PENTING!)
Sangat sering dipakai di Frontend (React, Vue, dll) untuk mengubah data mentah menjadi tampilan HTML.
```javascript
// Membuat array baru dimana semua nilai ditambah 5
const nilaiBonus = nilai.map((item) => item + 5);

console.log(nilaiBonus); // [65, 85, 95, 55, 80]
```

### C. .filter() - Menyaring data
Mengambil hanya data yang memenuhi syarat.
```javascript
// Mengambil nilai yang lulus (di atas 70)
const yangLulus = nilai.filter((item) => item > 70);

console.log(yangLulus); // [80, 90, 75]
```

## âš–ï¸ 4. Logika Percabangan (Control Flow)

Komputer perlu mengambil keputusan.

```javascript
let skor = 85;

if (skor >= 90) {
    console.log("Nilai A - Sangat Baik");
} else if (skor >= 75) {
    console.log("Nilai B - Baik");
} else {
    console.log("Nilai C - Cukup");
}
```

**Operator Logika:**
- `&&` (DAN): Semua syarat harus benar.
- `||` (ATAU): Salah satu syarat benar sudah cukup.
- `!` (TIDAK): Membalikkan kondisi.

```javascript
let punyaTiket = true;
let umur = 16;

if (punyaTiket && umur >= 17) {
    console.log("Boleh masuk bioskop");
} else {
    console.log("Ditolak masuk");
}
```
