# Materi 1.1: Membuat Website Merespon dengan DOM

## ğŸŒ³ Apa itu DOM?

**DOM** (Document Object Model) adalah jembatan antara **HTML** dan **JavaScript**.

Tanpa DOM, JavaScript tidak bisa mengubah apa pun di layar. Dengan DOM, JavaScript bisa:
1.  Mengganti teks.
2.  Mengubah warna atau ukuran elemen.
3.  Menghapus atau menambah kotak baru.

---

## ğŸ¯ Cara Memilih Elemen

Sebelum kita mengubah sesuatu, kita harus "menunjuk" elemen mana yang mau diubah. Ada 2 cara termudah:

### 1. Memilih berdasarkan ID (`getElementById`)
Paling cepat dan paling sering dipakai untuk elemen unik.
```javascript
const judul = document.getElementById("judul-utama");
```

### 2. Memilih seperti CSS (`querySelector`)
Bisa memilih berdasarkan tag, class, atau ID.
```javascript
const tombol = document.querySelector(".tombol-biru");
```

---

## âœï¸ Cara Mengubah Konten (Teks)

Setelah elemen "dipilih", kita bisa ganti isinya dengan `.textContent`.

**Contoh:**
```javascript
// HTML: <h1 id="sapaan">Halo</h1>

const teks = document.getElementById("sapaan");
teks.textContent = "Selamat Datang, User!";
```

---

## ğŸ¨ Cara Mengubah Tampilan (Style)

Kita bisa mengubah CSS langsung dari JavaScript menggunakan `.style`.

**Contoh:**
```javascript
const kotak = document.querySelector(".box");

kotak.style.backgroundColor = "red";  // Ubah warna latar
kotak.style.color = "white";          // Ubah warna tulisan
kotak.style.display = "none";         // Sembunyikan elemen
```
