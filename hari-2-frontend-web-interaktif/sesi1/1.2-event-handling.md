# Materi 1.2: Event Handling

## üéØ Apa itu Event?

**Event** adalah kejadian atau aksi yang terjadi di halaman web, seperti:
- User mengklik tombol
- User mengetik di input
- User mensubmit form
- Halaman selesai loading
- Mouse hover di atas elemen

**üí° Analogi:**
Event seperti **alarm** yang memberitahu JavaScript: "Hey, user baru saja klik tombol ini, lakukan sesuatu!"

---

## üîî Jenis-Jenis Event yang Sering Dipakai

### **Mouse Events:**
- `click` - Saat elemen diklik
- `dblclick` - Double click
- `mouseenter` - Mouse masuk area elemen
- `mouseleave` - Mouse keluar dari elemen
- `mousemove` - Mouse bergerak di atas elemen
- `mousedown` - Mouse button ditekan
- `mouseup` - Mouse button dilepas

### **Keyboard Events:**
- `keydown` - Tombol keyboard ditekan
- `keyup` - Tombol keyboard dilepas
- `keypress` - Tombol ditekan dan ditahan (deprecated)

### **Form Events:**
- `submit` - Form disubmit
- `input` - Nilai input berubah (realtime)
- `change` - Nilai input berubah (saat blur)
- `focus` - Input mendapat fokus
- `blur` - Input kehilangan fokus

### **Window Events:**
- `load` - Halaman selesai loading
- `DOMContentLoaded` - HTML selesai parsing (lebih cepat dari load)
- `resize` - Window di-resize
- `scroll` - User scroll halaman

## üé™ 3 Cara Menambahkan Event

### **1Ô∏è‚É£ Inline Event Handler** (TIDAK DISARANKAN)

**HTML:**
```html
<button onclick="alert('Button diklik!')">Klik Saya</button>
```

**‚ùå Kekurangan:**
- Mencampur HTML dengan JavaScript
- Sulit di-maintain
- Tidak bisa menambahkan multiple listeners

---

### **2Ô∏è‚É£ Property Event Handler**

**HTML:**
```html
<button id="myBtn">Klik Saya</button>
```

**JavaScript:**
```javascript
const btn = document.getElementById('myBtn');

btn.onclick = function() {
    alert('Button diklik!');
};

// Atau pakai arrow function
btn.onclick = () => {
    alert('Button diklik!');
};
```

**‚ö†Ô∏è Kekurangan:**
- Hanya bisa 1 handler per event
- Jika assign lagi, yang lama akan ditimpa

---

### **3Ô∏è‚É£ `addEventListener()` ‚≠ê BEST PRACTICE**

**JavaScript:**
```javascript
const btn = document.getElementById('myBtn');

btn.addEventListener('click', function() {
    alert('Button diklik!');
});

// Bisa tambah multiple listeners
btn.addEventListener('click', () => {
    console.log('Handler kedua!');
});
```

**‚úÖ Keuntungan:**
- Bisa multiple event listeners
- Bisa di-remove dengan `removeEventListener`
- Lebih clean & modular
- Bisa control phase (capture/bubble)

**Sintaks:**
```javascript
element.addEventListener(eventType, callbackFunction, options);
```

## üéÆ Event Object

Setiap event handler menerima **event object** sebagai parameter.

**JavaScript:**
```javascript
const btn = document.querySelector('#myBtn');

btn.addEventListener('click', (event) => {
    console.log(event);  // Object berisi info tentang event
    
    console.log(event.type);         // "click"
    console.log(event.target);       // Elemen yang diklik
    console.log(event.currentTarget); // Elemen yang dipasangi listener
    console.log(event.clientX);      // Posisi X mouse
    console.log(event.clientY);      // Posisi Y mouse
});
```

### **Property Penting dari Event Object:**

| Property | Deskripsi |
|----------|-----------|
| `event.type` | Jenis event ("click", "keydown", dll) |
| `event.target` | Elemen yang memicu event |
| `event.currentTarget` | Elemen yang punya event listener |
| `event.preventDefault()` | Mencegah default behavior |
| `event.stopPropagation()` | Stop event bubbling |
| `event.clientX / clientY` | Posisi mouse |
| `event.key` | Tombol keyboard yang ditekan |

## üñ±Ô∏è Mouse Events - Contoh Praktis

### **Example 1: Click Counter**

**HTML:**
```html
<button id="clickBtn">Click Me!</button>
<p>Clicked: <span id="count">0</span> times</p>
```

**JavaScript:**
```javascript
const btn = document.getElementById('clickBtn');
const countDisplay = document.getElementById('count');
let clickCount = 0;

btn.addEventListener('click', () => {
    clickCount++;
    countDisplay.textContent = clickCount;
});
```

---

### **Example 2: Mouse Position Tracker**

**HTML:**
```html
<div id="tracker">
    <p>Mouse X: <span id="xPos">0</span></p>

</div>
```

```javascript
const xDisplay = document.getElementById('xPos');
const yDisplay = document.getElementById('yPos');

document.addEventListener('mousemove', (e) => {
    xDisplay.textContent = e.clientX;
    yDisplay.textContent = e.clientY;

```

---

### **Example 3: Hover Effect**

**HTML:**
```html
<div id="box" style="width:200px; height:200px; background:blue;"></div>
```

**JavaScript:**
```javascript

});

    box.style.backgroundColor = 'blue';
    box.style.transform = 'scale(1)';
});
```

### **Example 3: WASD Movement (Game-like)**

**HTML:**
```html
<div id="player" style="position:absolute; width:50px; height:50px; background:red; top:100px; left:100px;"></div>
<p>Gunakan WASD untuk bergerak</p>
```

**JavaScript:**
```javascript
const player = document.getElementById('player');
let posX = 100;
let posY = 100;
const speed = 10;

document.addEventListener('keydown', (e) => {
    switch(e.key.toLowerCase()) {
        case 'w':
            posY -= speed;
            break;
        case 's':
            posY += speed;
            break;
        case 'a':
            posX -= speed;
            break;
        case 'd':
            posX += speed;
            break;
    }
    
    player.style.top = posY + 'px';
    player.style.left = posX + 'px';
});
```

### **Example 3: Form Submit Handler**

**HTML:**
```html
<form id="loginForm">
    <input type="text" id="username" placeholder="Username" required>
    <input type="password" id="password" placeholder="Password" required>
    <button type="submit">Login</button>
</form>
```

**JavaScript:**
```javascript
const form = document.getElementById('loginForm');

form.addEventListener('submit', (e) => {
    e.preventDefault();  // Mencegah page reload
    
    const username = document.getElementById('username').value;
    const password = document.getElementById('password').value;
    
    console.log('Username:', username);
    console.log('Password:', password);
    
    // Kirim ke server (nanti akan dipelajari)
    alert(`Login sebagai: ${username}`);
});
```

**üí° Penting:** `e.preventDefault()` mencegah form reload halaman!

## üéØ Event Delegation (Advanced)

Event delegation adalah teknik menangani event di **parent element** untuk semua children.

**Kenapa?** Lebih efisien jika punya banyak child elements.

**Contoh:**

**HTML:**
```html
<ul id="todoList">
    <li>Task 1 <button class="delete">‚ùå</button></li>
    <li>Task 2 <button class="delete">‚ùå</button></li>
    <li>Task 3 <button class="delete">‚ùå</button></li>
</ul>
```

**‚ùå Cara Tidak Efisien:**
```javascript
const deleteButtons = document.querySelectorAll('.delete');
deleteButtons.forEach(btn => {
    btn.addEventListener('click', (e) => {
        e.target.parentElement.remove();
    });
});
```

**‚úÖ Cara Efisien (Event Delegation):**
```javascript
const todoList = document.getElementById('todoList');

todoList.addEventListener('click', (e) => {
    if (e.target.classList.contains('delete')) {
        e.target.parentElement.remove();
    }
});
```

**Keuntungan:**
- Hanya 1 event listener (bukan 3 atau lebih)
- Otomatis bekerja untuk elemen baru yang ditambahkan
