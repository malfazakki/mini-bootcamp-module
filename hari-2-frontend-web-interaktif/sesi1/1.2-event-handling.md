# Materi 1.2: Menjalankan Perintah dengan Klik (Events)

## âš¡ Apa itu Event?

**Event** adalah sinyal bahwa sesuatu telah terjadi di website. JavaScript mendengarkan sinyal ini untuk menjalankan perintah.

Contoh event yang paling sering kita pakai:
1.  **click**: Saat tombol atau elemen diklik.
2.  **input**: Saat kita mengetik sesuatu di kotak isian.

---

## ðŸ‘‚ Cara Mendengarkan Event (`addEventListener`)

Gunakan perintah ini agar JavaScript "siaga" menunggu aksi dari user.

**Contoh 1: Klik Tombol**
```javascript
const tombol = document.getElementById("klik-saya");

tombol.addEventListener("click", function() {
    alert("Tombol Berhasil Diklik!");
});
```

**Contoh 2: Mengetik di Input (Real-time)**
```javascript
const kotakInput = document.getElementById("nama");

kotakInput.addEventListener("input", function() {
    console.log("Kamu sedang mengetik: " + kotakInput.value);
});
```

---

## ðŸŽ¯ Kenapa Event Penting?

Tanpa event, website kita hanya akan jadi gambar diam. Event membuat website:
- Bisa membuka menu saat diklik.
- Bisa menghitung angka saat tombol ditekan.
- Bisa memberikan info otomatis saat user mengisi form.
