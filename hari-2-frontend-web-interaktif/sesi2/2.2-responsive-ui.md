# Materi 2.2: Responsive UI (Mobile & Desktop)

## üì± Apa itu Responsive Web Design?

**Responsive Web Design** adalah teknik membuat website yang tampil bagus dan berfungsi baik di **semua ukuran layar**‚Äîdari smartphone hingga desktop.

**üí° Analogi:**
Seperti baju yang bisa menyesuaikan ukuran tubuh pemakai secara otomatis!

**Kenapa Penting?**
- ‚úÖ >60% traffic internet dari mobile
- ‚úÖ Google prioritaskan mobile-friendly sites (SEO)
- ‚úÖ Meningkatkan user experience
- ‚úÖ Lebih efisien (1 website untuk semua device)

---

## üìê Breakpoints (Ukuran Layar Standar)

| Device | Width | Breakpoint |
|--------|-------|------------|
| üì± Mobile (Portrait) | 320px - 480px | `max-width: 480px` |
| üì± Mobile (Landscape) | 481px - 767px | `max-width: 767px` |
| üì± Tablet | 768px - 1024px | `max-width: 1024px` |
| üíª Desktop | 1025px - 1200px | `min-width: 1025px` |
| üñ•Ô∏è Large Desktop | 1201px+ | `min-width: 1201px` |

**üí° Standar Umum:**
- Mobile: `< 768px`
- Tablet: `768px - 1024px`
- Desktop: `> 1024px`

---

## üìã Viewport Meta Tag (WAJIB!)

Tanpa tag ini, website tidak akan responsive di mobile!

```html
<head>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
```

**Penjelasan:**
- `width=device-width` ‚Üí Lebar mengikuti lebar device
- `initial-scale=1.0` ‚Üí Zoom level awal 100%

## üé® Media Queries - Inti Responsive Design

**Media query** adalah teknik CSS untuk menerapkan style berbeda berdasarkan kondisi tertentu (biasanya ukuran layar).

### **Sintaks Dasar:**

```css
@media (kondisi) {
    /* CSS yang hanya berlaku jika kondisi terpenuhi */
}
```

---

### **Example 1: Mobile First Approach** ‚≠ê RECOMMENDED

```css
/* Default style (untuk mobile) */
.container {
    width: 100%;
    padding: 10px;
}

.box {
    width: 100%;
    margin-bottom: 20px;
}

/* Tablet */
@media screen and (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
    }
    
    .box {
        width: 48%;
        float: left;
        margin-right: 2%;
    }
}

/* Desktop */
@media screen and (min-width: 1024px) {
    .container {
        width: 980px;
    }
    
    .box {
        width: 30%;
    }
}
```

**üí° Mobile First:** CSS default untuk mobile, tambahkan style untuk screen lebih besar.

---

### **Example 2: Desktop First Approach**

```css
/* Default style (untuk desktop) */
.container {
    width: 1200px;
    margin: 0 auto;
}

.box {
    width: 30%;
    float: left;
}

/* Tablet */
@media screen and (max-width: 1024px) {
    .container {
        width: 100%;
        padding: 20px;
    }
    
    .box {
        width: 48%;
    }
}

/* Mobile */
@media screen and (max-width: 768px) {
    .box {
        width: 100%;
        float: none;
    }
}
```

**üí° Desktop First:** CSS default untuk desktop, override untuk screen lebih kecil.

## üìè Unit Relatif vs Absolut

### **Absolut (Fixed Size):**
```css
div {
    width: 300px;     /* Selalu 300px */
    font-size: 16px;  /* Selalu 16px */
}
```

**‚ùå Masalah:** Tidak responsive!

---

### **Relatif (Flexible Size):**

#### **1Ô∏è‚É£ Percentage (%)**
Relatif terhadap **parent element**.

```css
.container {
    width: 80%;  /* 80% dari lebar parent */
}
```

---

#### **2Ô∏è‚É£ vw & vh (Viewport Width & Height)**
Relatif terhadap **ukuran viewport**.

```css
.hero {
    width: 100vw;   /* 100% lebar viewport */
    height: 100vh;  /* 100% tinggi viewport */
}

.title {
    font-size: 5vw;  /* 5% dari lebar viewport */
}
```

**üí° Contoh:** Jika viewport 1000px, maka `10vw` = 100px.

---

#### **3Ô∏è‚É£ rem & em**
**rem** ‚Üí Relatif terhadap **root font-size** (biasanya 16px)
**em** ‚Üí Relatif terhadap **parent font-size**

```css
html {
    font-size: 16px;  /* 1rem = 16px */
}

h1 {
    font-size: 2rem;   /* 32px (2 x 16px) */
}

p {
    font-size: 1rem;   /* 16px */
}

.small {
    font-size: 0.875rem;  /* 14px */
}
```

**üí° Rekomendasi:** Gunakan `rem` untuk font size!

## üéØ Responsive Images

### **1Ô∏è‚É£ Max-width 100%**
Gambar tidak akan lebih besar dari containernya.

```css
img {
    max-width: 100%;
    height: auto;  /* Menjaga aspect ratio */
}
```

---

### **2Ô∏è‚É£ Picture Element (Multiple Sources)**

```html
<picture>
    <!-- Untuk mobile -->
    <source media="(max-width: 767px)" srcset="image-mobile.jpg">
    
    <!-- Untuk tablet -->
    <source media="(max-width: 1024px)" srcset="image-tablet.jpg">
    
    <!-- Default untuk desktop -->
    <img src="image-desktop.jpg" alt="Responsive Image">
</picture>
```

---

### **3Ô∏è‚É£ Background Image**

```css
.hero {
    background-image: url('desktop-bg.jpg');
    background-size: cover;
    background-position: center;
    height: 500px;
}

@media (max-width: 768px) {
    .hero {
        background-image: url('mobile-bg.jpg');
        height: 300px;
    }
}
```

## üé® Responsive Typography

```css
/* Base font size */
html {
    font-size: 16px;  /* Desktop */
}

h1 {
    font-size: 2.5rem;  /* 40px */
}

p {
    font-size: 1rem;    /* 16px */
    line-height: 1.6;
}

/* Mobile */
@media (max-width: 768px) {
    html {
        font-size: 14px;  /* Semua ukuran rem mengecil otomatis! */
    }
    
    h1 {
        font-size: 2rem;  /* 28px (2 x 14px) */
    }
}
```

**üí° Clamp untuk Fluid Typography:**
```css
h1 {
    font-size: clamp(1.5rem, 5vw, 3rem);
    /* Min: 1.5rem, Ideal: 5vw, Max: 3rem */
}
```

## üèóÔ∏è Responsive Navigation

### **Example: Hamburger Menu**

**HTML:**
```html
<nav class="navbar">
    <div class="nav-brand">MyWebsite</div>
    
    <!-- Hamburger Icon -->
    <button class="nav-toggle" id="navToggle">
        <span></span>
        <span></span>
        <span></span>
    </button>
    
    <!-- Menu Links -->
    <ul class="nav-menu" id="navMenu">
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#services">Services</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>
```

**CSS:**
```css
/* Desktop style */
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 15px 50px;
    background-color: #333;
}

.nav-brand {
    color: white;
    font-size: 24px;
    font-weight: bold;
}

.nav-toggle {
    display: none;  /* Sembunyikan di desktop */
}

.nav-menu {
    display: flex;
    list-style: none;
    gap: 30px;
}

.nav-menu a {
    color: white;
    text-decoration: none;
    transition: color 0.3s;
}

.nav-menu a:hover {
    color: #3498db;
}

/* Mobile style */
@media (max-width: 768px) {
    .nav-toggle {
        display: block;
        background: none;
        border: none;
        cursor: pointer;
        padding: 0;
    }
    
    .nav-toggle span {
        display: block;
        width: 25px;
        height: 3px;
        background: white;
        margin: 5px 0;
        transition: all 0.3s;
    }
    
    .nav-menu {
        position: fixed;
        left: -100%;  /* Sembunyikan di kiri */
        top: 70px;
        flex-direction: column;
        background-color: #333;
        width: 100%;
        text-align: center;
        transition: left 0.3s;
        padding: 20px 0;
    }
    
    .nav-menu.active {
        left: 0;  /* Muncul dari kiri */
    }
}
```

**JavaScript:**
```javascript
const navToggle = document.getElementById('navToggle');
const navMenu = document.getElementById('navMenu');

navToggle.addEventListener('click', () => {
    navMenu.classList.toggle('active');
});
```

## üéØ Yang Harus Dikuasai:

- ‚úÖ Viewport meta tag
- ‚úÖ Media queries (`@media`)
- ‚úÖ Mobile first vs Desktop first approach
- ‚úÖ Unit relatif (%, vw, vh, rem, em)
- ‚úÖ Responsive images (`max-width: 100%`)
- ‚úÖ Flexbox & Grid untuk responsive layout
- ‚úÖ Test responsive di berbagai ukuran

---

**üí° Tips Best Practices:**
- Gunakan **Mobile First** approach
- Test di **real device**, bukan hanya DevTools
- Gunakan **rem** untuk font sizes
- Gunakan **Grid atau Flexbox** untuk layout
- **Touch targets** minimal 48x48px untuk mobile
- Hindari horizontal scroll di mobile! üöÄ
