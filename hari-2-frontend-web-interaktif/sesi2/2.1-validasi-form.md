# Materi 2.1: Validasi Form Sederhana

## âœ… Kenapa Harus Validasi?

Validasi adalah cara kita mengecek apakah data yang diisi user sudah benar sebelum dikirim. Misalnya:
- Jangan sampai nama dikosongkan.
- Jangan sampai password terlalu pendek.

---

## ğŸ“ 1. Validasi Otomatis (HTML5)

Cara termudah dan tanpa ngoding JavaScript sama sekali. Cukup tambah atribut di tag `<input>`.

- `required`: Wajib diisi.
- `minlength="5"`: Minimal 5 karakter.
- `type="email"`: Harus format email.

**Contoh:**
```html
<input type="text" required minlength="3" placeholder="Masukkan Nama">
```

---

## ğŸ’» 2. Validasi dengan JavaScript

Kadang kita ingin kontrol lebih, misalnya memunculkan peringatan khusus.

**Contoh Logika:**
```javascript
const tombol = document.getElementById("kirim");
const inputNama = document.getElementById("nama");

tombol.addEventListener("click", function() {
    if (inputNama.value === "") {
        alert("Maaf, nama tidak boleh kosong!");
    } else if (inputNama.value.length < 3) {
        alert("Nama minimal harus 3 huruf!");
    } else {
        alert("Terima kasih, data terkirim!");
    }
});
```
